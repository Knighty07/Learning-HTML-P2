Steps in building an Image on a webpage

  • Pick an Image (careful of the file extention in the url)
  • Record URL, cite it, connect for date access, and record original dimensions
  • Use blockquote and comment as approoperiate
  • Use img-tag with attributes src,alt, title, and style with width-height
